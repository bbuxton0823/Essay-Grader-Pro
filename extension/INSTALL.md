# Writing Assistant Pro - Installation Guide

## Quick Install (5 minutes)

This extension installs locally on YOUR computer only. No one else can see it, detect it, or know it exists.
No IT approval needed. No store. No trace.

---

## Step 1: Download the Extension

1. Download this entire `essay-grader-extension` folder to your computer
2. Put it somewhere safe (e.g., `Documents/MyTools/essay-grader-extension`)
3. **DO NOT delete this folder** - Chrome needs it to run the extension

---

## Step 2: Open Chrome Extensions

1. Open Google Chrome
2. Type `chrome://extensions` in the address bar and press Enter
3. You'll see the Extensions management page

---

## Step 3: Enable Developer Mode

1. Look for the **"Developer mode"** toggle in the top-right corner
2. Click it to turn it ON (toggle should be blue)

---

## Step 4: Load the Extension

1. Click the **"Load unpacked"** button (top-left area)
2. Navigate to the `essay-grader-extension` folder you downloaded
3. Select the folder and click "Open" or "Select Folder"
4. The extension will appear in your list

---

## Getting an API Key

### Option A: Anthropic (Claude) - Recommended
1. Go to https://console.anthropic.com/
2. Create an account or sign in
3. Navigate to "API Keys"
4. Create a new key
5. Copy it (starts with `sk-ant-`)

### Option B: OpenAI (GPT-4)
1. Go to https://platform.openai.com/
2. Create an account or sign in
3. Navigate to "API Keys"
4. Create a new secret key
5. Copy it (starts with `sk-`)

**Note:** Both services charge per use. Typical cost is $0.01-0.05 per essay graded.

## How to Use

### On Schoology:
1. Navigate to a student's essay submission
2. Press `Ctrl+Shift+G` (or `Cmd+Shift+G` on Mac)
3. The grading sidebar will open
4. Click "Extract from Page" or paste the essay text
5. Click "Grade Essay"
6. Review results and copy the grade to Schoology

### On Google Classroom:
1. Open a student's submission
2. Press `Ctrl+Shift+G` (or `Cmd+Shift+G` on Mac)
3. The grading sidebar will open
4. Click "Extract from Page" or paste the essay text
5. Click "Grade Essay"
6. Review results and copy the grade to Classroom

## Troubleshooting

### "Extension not working on Schoology/Classroom"
- Make sure you're on a submission/grading page
- Try refreshing the page
- Check that the extension is enabled in `chrome://extensions`

### "API key not working"
- Make sure you copied the full key
- Check that you selected the correct provider
- Verify your API account has credits

### "Can't extract essay text"
- Some submissions (especially Google Docs) may require manual copy/paste
- The extension will tell you if manual extraction is needed

### "Extension disappeared"
- Go to `chrome://extensions`
- Make sure Developer Mode is ON
- Click "Load unpacked" and re-select the folder
